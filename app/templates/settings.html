How To Feed Prompts

  1. Use .claude/settings.example.json with python scripts/install_hooks.py --client claude for Claude Code.
  2. Use .codex/hooks.example.json with python scripts/install_hooks.py --client codex for Codex.
  3. Use the API endpoint POST /api/prompts from any custom hook or wrapper.
  4. Drop .jsonl or .txt files into {{ settings.import_dir }}.
  5. Use the manual capture page when you just want to paste a prompt yourself.
